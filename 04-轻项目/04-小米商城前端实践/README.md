# README

## ★为何选择 Vue？

Vue.js灵活易学而且功能强大，是最流行的且性价比也最高的前端框架，国内绝大部分公司都在使用，它的生态环境仍在生长并且它已经拥有构建各种应用所需的一切

## ★你能学到什么？

1. 在实战中学习和应用Vue全家桶：Vue、Vuex、Vue-router、Vue-cli、Vue-devtools、Vue-loader
2. 通过重构小米商城（移动端），理解vue在实际业务场景中的注意事项和技巧，深入理解组件化和数据驱动常见问题
